# Project Title (Update)

My project uses recursion to find the factorial of an integer.

## Instructions for Build and Use

Steps to build and/or run the software:

1. Download Erlang
2. Run in the console

Instructions for using the software:

1. Put the number you want into the function.
2. Compile the program in the terminal.
3. Call the function in the terminal.
